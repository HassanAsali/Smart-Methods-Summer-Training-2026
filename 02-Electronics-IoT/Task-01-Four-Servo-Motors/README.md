# Task 01 - Four Servo Motors

## Project Description

In this task, I used an Arduino Uno to control four servo motors in Tinkercad.

The four servo motors sweep from 0 to 180 degrees and return to 0 degrees. This movement takes about 2 seconds. After that, all the servo motors move to 90 degrees and stay there.

## Components

- Arduino Uno
- 4 Micro Servo Motors
- Small Breadboard
- Jumper Wires

## Connections

The servo signal wires are connected to Arduino pins 2, 4, 7, and 8.

All servo motors are connected to 5V and GND using the breadboard.
